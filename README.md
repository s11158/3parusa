# Сайт 3parusa.ru

Статическая копия сайта «Туры на парусной яхте», перенесённая с Tilda на GitHub Pages.

- `index.html` — главная (и единственная) страница.
- `assets/` — все стили, скрипты, шрифты и изображения (локальные копии).
- `CNAME` — привязка к домену `3parusa.ru`.
- Формы заявок отправляются через [Formspree](https://formspree.io) на e-mail владельца.

## Как обновлять
Отредактируйте нужный файл и закоммитьте — GitHub Pages пересоберёт сайт автоматически.
